# BackEnd_todoList

> todolist의 backEnd이다.

## 사용스킬

|  스킬   | 내용                                                                                                             |
| :-----: | ---------------------------------------------------------------------------------------------------------------- |
| Express | nodeJS의 framework로 node보다 더 간결하고 json으로 데이터를 주고받아 ReactJS와 VueJS와의 데이터상호작용이 편하다 |
| Router  | router를 이용하여 rest API 구분하여 코딩할 수 있다.                                                              |
|  babel  | ES6이상의 문법을 읽지못하는 Express를 위해 babel을 통해 ES6이상을 ES5로 변환해 준다.                             |
